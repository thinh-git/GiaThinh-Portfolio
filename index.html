Mục tiêu nghề nghiệp

  • Nắm vững React & Node.js
  • Học Cloud Computing (AWS)
  • Tham gia dự án thực tế
  • Xây dựng portfolio chuyên nghiệp

Dài hạn (3-5 năm)

  • Trở thành Senior Developer
  • Dẫn dắt team phát triển
  • Đóng góp open source
  • Khởi nghiệp công nghệ

  • Code chất lượng cao
  • Học hỏi liên tục

  • Đổi mới sáng tạo
